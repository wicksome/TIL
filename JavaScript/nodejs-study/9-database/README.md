# mysql Database

**설치**

1. Homebrew 설치
2. mysql 설치: `$ brew install mysql`
3. mysql 실행: `$ mysql.server restart`
4. 접속
   1. 익명 사용자 접속: `$ mysql -u root`
   2. root 비밀번호 설정: `$ mysq_secure_installation`
