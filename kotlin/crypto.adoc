== AES256

* aes-256-cbc로 생성하면 64byte 키값이 생성됨
+
[source]
----
salt=5B1CD0A0E6F91535
key=1F3CDAD3FDD397162A8431B24D67B65C7C4A0277BC07223C19B7CD711F4376F5
iv =BF9B3C014A9061CB7F33FF3451023ABE
----
