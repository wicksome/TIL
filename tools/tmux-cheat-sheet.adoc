== Panes

|===
|Shortcut |Purpose

|Ctrl + b " | Split pane horizontally `⊟`
|Ctrl + b % |Split pane vertically `⎅`
|Ctrl + b { |Move pane left
|Ctrl + b } |Move pane right
|Ctrl + b ← ↑ → ↓ |Move pane
|Ctrl + b x |Close pane
|===

`:setw synchronize-panes`
`:setw synchronize-panes off`
